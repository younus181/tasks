TAGS

Basic Structure Of Tags

< tag> content < /tag>

Opening tag:< tag>
Closing tag: < /tag>
Content: Text or elements inside the tags

Types of tags

1.Paired tags:These tags have both an opening and a closing tag.
Ex:< h1> to < h6>,< p>,< b>,< ul>
2.Unpaired tags (or) Self Closing Tags:These tags do not have a closing tags and they do not contain any content.
Ex: < br>,< img>

Types Of Elements

1.Block level Elements
That default start on a new line and occupy the full width available to them.
Ex:< h1 to h6>,< p>,< ul>
2.Inline Block Elements
Which occupies only required space and doesn't create a new line
Ex:img,iframe, input,button
Inline Elements
Element that doesn't start on a new line and only occupy as much width as necessary
Eg:span,b,strong,i
